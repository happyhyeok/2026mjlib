🏃 엔트리 코딩: 달리기 시합!

준비 1단계: 경기장과 선수들
미션: 시합을 할 친구들과 멋진 잔디밭을 준비해 주세요.

1. 엔트리봇을 삭제해요.

2. 배경 오브젝트를 추가해요. 오브젝트 추가하기를 클릭하여, 배경에서 잔디밭을 찾아서 추가하세요.

3. 선수들을 데려와요. 오브젝트 추가하기를 클릭하여, 동물에서 '고양이', '곰', '강아지'를 추가해요.

4. 선수들이 너무 크면 시합하기 힘들니 크기를 80으로 줄여주세요.

핵심 2단계: 고양이의 '무한 반복' 훈련
미션: 고양이가 스스로 다리를 움직이며 달려가게 만드세요.
💡 어떻게 만들 수 있을까요?
1. 고양이에게 달리기를 시작하라고 신호를 알려줘야 해요. → [시작]꾸러미 [시작하기 버튼을 클릭했을 때]

2. 고양이는 오른쪽으로 달려가야 해요. 엔트리에서 오른쪽 가는 것은 X좌표를 커지게 만드는 거예요. → [움직임]꾸러미 [x 좌표를 2만큼 바꾸기]

3. 고양이가 걷는 모양, 뛰는 모양으로 계속 바꿔줘야 진짜 움직이는 것 같아요. → [생김새]꾸러미 [다음 모양으로 바꾸기]

4. 고양이가 오른쪽으로 한 번만 가면 끝까지 갈 수 없어요. 끝까지 가려면, 오른쪽으로 가는 동작을 계속 반복해야 해요. → [흐름]꾸러미 [계속 반복하기]

5. 컴퓨터가 너무 눈 깜빡할 새 달리지 않게 천천히 속도를 조절해 줘요. → [흐름]꾸러미 [0.1초 기다리기]
시작하기 버튼을 클릭했을 때
   ┗ 계속 반복하기
         ┗ 다음 모양으로 바꾸기
         ┗ x 좌표를 2만큼 바꾸기
         ┗ 0.1초 기다리기
연습 3단계: 곰 아저씨는 전력 질주!
미션: 곰 아저씨가 고양이보다 조금 더 빨리 달려가게 만드세요.
💡 다시 조립하며 기억하기
1. 곰 아저씨를 클릭하고 코딩창을 확인하세요.

2. 곰 아저씨에게도 시작 신호를 먼저 주세요. → [시작]꾸러미 [시작하기 버튼을 클릭했을 때]

3. 곰 아저씨도 고양이처럼 오른쪽으로 달려가는데, 고양이보다 더 빨리 달려가야 해요. (고양이는 2, 곰 아저씨는 10) → [움직임]꾸러미 [x 좌표를 10만큼 바꾸기]

4. 곰 아저씨도 고양이처럼 모양을 계속 바꿔줘서, 진짜 움직이는 것처럼 보이게 해요.

5. 곰 아저씨도 끝까지 달리도록 반복시켜 주세요.

6. 컴퓨터가 너무 눈 깜빡할 새 달리지 않게 천천히 속도를 조절해 줘요.

응원 4단계: 강아지는 나의 응원이 필요해!
미션: 강아지는 마우스를 클릭해서, 우리가 뛰게 만들어요.
💡 어떻게 만들 수 있을까요?
1. 강아지를 클릭하고 코딩창을 확인하세요.

2. 강아지에게 마우스를 클릭해서 뛰라고 신호를 보내요. → [시작]꾸러미 [마우스를 클릭했을 때]

3. 응원을 받았으니 한 번에 아주 멀리 슝~ 하고 오른쪽으로 날아가게 해요. → [움직임]꾸러미 [x 좌표를 30만큼 바꾸기]

4. 슝하고 날아갈 때, 모양을 바꿔서 신나게 달리는 모습을 표현해요. → [생김새]꾸러미 [다음 모양으로 바꾸기]
마우스를 클릭했을 때
   ┗ x 좌표를 30만큼 바꾸기
   ┗ 다음 모양으로 바꾸기
레벨 업 더 재미있게 만들어 봐요!
🚀 확장 1: 누가 1등인지 말해주기!
1. 고양이, 곰아저씨, 강아지에게 모두 블록을 추가해요.

2. 오른쪽 벽에 닿으면 1등이라고 말하고, 다른 선수들에게도 알려줘야 해요. [시작]꾸러미 [신호 만들기]에서 "1등 알리기" 신호를 먼저 만드세요.

3. 오른쪽에 닿았는지 물어봐야 해요. → [흐름]꾸러미 [만일 참이라면] 속에 [판단]꾸러미 [오른쪽벽에 닿았는가?]를 넣으세요.

4. 벽에 닿았다면 기쁘게 외쳐요. → [생김새]꾸러미 [내가 1등이다!를 4초 동안 말하기]

5. 다른 친구들에게도 소식을 전해요. → [시작]꾸러미 [1등 알리기 신호 보내기]

6. 언제 벽에 닿을지 모르니 [계속 반복하기] 울타리 안에 이 명령들을 넣어주세요.

7. 다른 친구가 1등 신호를 보내면 달리기를 멈춰요. → [시작]꾸러미 [1등 알리기 신호를 받았을 때] 아래에 [흐름]꾸러미 [자신의 다른 코드 멈추기]를 붙여주세요.
🚀 확장 2: 미끌미끌 바나나 장애물
1. 오브젝트 추가하기에서 바나나를 가져와 달리기 트랙 중간에 두세요.

2. 선수들이 바나나를 밟았는지 계속 감시해야 해요. → [흐름]꾸러미 [만일 참이라면] 울타리를 준비하세요.

3. 바나나에 닿았는지 확인하는 블록을 넣어요. → [판단]꾸러미 [바나나에 닿았는가?]

4. 닿았다면 뒤로 미끄러져요. → [움직임]꾸러미 [x 좌표를 -50만큼 바꾸기]

5. 이 모든 과정도 달리기가 끝날 때까지 [계속 반복하기] 안에 있어야 해요!
🚀 확장 3: 관중석의 환호성 소리 추가
1. 배경을 클릭하고 코딩창을 확인하세요.

2. [소리] 탭을 클릭해서 '환호성'이나 '박수' 소리를 추가하세요.

3. 시합이 시작되자마자 소리가 나게 해요. → [시작]꾸러미 [시작하기 버튼을 클릭했을 때]

4. 시합 내내 소리가 들려야 신나겠죠? → [흐름]꾸러미 [계속 반복하기] 울타리를 가져오세요.

5. 소리를 끝까지 들려줘요. → [소리]꾸러미 [소리 환호성 재생하기]를 반복문 안에 넣어주세요.
